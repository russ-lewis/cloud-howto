What is Instance Metadata?

In EC2, the "instance metadata" is information that can be accessed, from the EC2 instance, using HTTPS requests to a private IP address. You should be aware that some (all?) of this data is open to all users running on that instance; thus, you just not use it to store private data, if you plan to have untrusted users sharing the machine.
